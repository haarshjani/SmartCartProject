# Customer Purchasing Behavior Analysis using Unsupervised Machine Learning

This project implements customer segmentation using **K-Means** and **Agglomerative Clustering** to identify distinct customer groups based on purchasing patterns. **PCA** is applied for dimensionality reduction and cluster visualization, while **Anomaly Detection** helps uncover unusual customer behavior and outliers. 

The analysis explores key business metrics such as online vs. offline purchases, purchase amount, customer count, and complaint statistics. The generated insights support data-driven marketing strategies, customer targeting, and business decision-making.

## Technologies Used

* Python
* Pandas & NumPy
* Scikit-learn
* PCA
* K-Means Clustering
* Agglomerative Clustering
* Matplotlib & Seaborn
